# 🏥 Análisis de Costos Clínicos y Facturación Médica

Este proyecto presenta una solución integral para el análisis de datos de facturación médica, aplicando técnicas de limpieza (ETL), análisis exploratorio de datos (EDA), reducción de dimensionalidad (PCA) y una interfaz interactiva pública.

## 📂 Estructura del Repositorio
Cumpliendo con los estándares de desarrollo, el proyecto está organizado de la siguiente manera:

* `data/raw/`: Contiene el dataset original sin modificaciones (`reporte_clinica.csv`).
* `data/processed/`: Contiene el dataset final utilizado durante el análisis limpio de nulos y duplicados.
* `notebooks/`: Contiene el cuaderno de Google Colab (`.ipynb`) con el desarrollo ordenado de las etapas.
* `app/`: Contiene la aplicación pública desarrollada en Streamlit (`Home.py`).
* `reports/`: Contiene el informe final en formato PDF.
* `logs/`: Registro de transformaciones y eventos del proceso.
* `requirements.txt`: Declaración de las librerías necesarias para ejecutar el proyecto.
